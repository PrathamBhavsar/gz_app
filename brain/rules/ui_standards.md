# RULE: UI STANDARDS
> **Enforcement Level**: CRITICAL

1. **Design Tokens:** Use established App Colors and Tokens. No raw `Color(0xFF...)`. 
2. **Screenutil Sizing:** EVERY size must use a screenutil suffix (`.r`, `.sp`, `.w`). Raw pixel sizes are forbidden.
3. **Typography:** Never omit `fontFamily` in TextStyles. Use tokens.
